# ProgramavimasTP
# V01
Sukuriama struktūra, kurioje bus saugomi studento duomenys. Ranka įvedami duomenys - studentų vardas, pavardė, pažymiai ir egzamino pažymys. Apskaičiuojami pažymių vidurkiai ir mediana (medianos skaičiavimui naudojama atskira funkcija). Tuomet apskaičiuojami galutiniai pažymiai naudojantis gautais vidurkiais ir medianomis. Visi atsakymai išvedami į ekraną.
